# RAN-slicing use case

This folder contains the curated RAN-slicing scripts/notebooks retained for provenance and public explanation of the SIA workflow.

The underlying dataset is confidential, so this public repo intentionally ships only:

- the runnable scripts/notebooks,
- the generated public result artifacts used in the paper workflow, and
- documentation describing how the SIA analysis was applied.

If you need the original dataset, contact the Northeastern University coauthors listed in the paper: Leonardo Bonati, Salvatore D'Oro, Michele Polese, and Tommaso Melodia (`{l.bonati, s.doro, m.polese, t.melodia}@northeastern.edu`).
