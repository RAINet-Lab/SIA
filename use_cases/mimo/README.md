# Massive-MIMO use case

This folder contains curated Massive-MIMO notebooks and analysis entrypoints retained from the historical SIA workflow.

The public repo includes the analysis code and paper references, but not the original training traces. The environment used by this workflow is tied to *A Deep Reinforcement Learning-Based Resource Scheduler for Massive MIMO Networks* (Qing An, Santiago Segarra, Chris Dick, Ashutosh Sabharwal, and Rahman Doost-Mohammady, IEEE TMLCN 2023).

If you need the training traces, contact the original paper authors, especially the Rice University group associated with that scheduler paper. Generated visualizations and large intermediate outputs are intentionally excluded from git.
