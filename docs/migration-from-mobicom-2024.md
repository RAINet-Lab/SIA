# Migration from `mobicom-2024`

The original private research repo mixed reusable code, raw data, generated artifacts, exploratory notebooks, and venue-specific history.

This public repo keeps:

- the shared SIA core modules from `scripts/`
- curated ABR agents from `pensive/`
- curated MIMO and RAN-slicing notebooks

This public repo excludes by default:

- raw datasets
- generated figures and PDFs
- cached models and logs
- editor metadata
